# Frontend - Student Feedback Analyzer

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Components

- `Dashboard` - Main dashboard with overview
- `FeedbackForm` - Submit new feedback
- `FeedbackList` - View all feedbacks
- `Analytics` - Analytics and visualizations
- `UrgentAlerts` - Urgent alerts view
